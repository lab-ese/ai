2. ALGORITHM IN STEPS:
Step 1: Define a vocabulary list of known correctly spelled words.
Step 2: Accept a target word from the user.
Step 3: Perform a direct lookup. If the target word is in the vocabulary, it's correct. Stop.
Step 4: If misspelled, filter the vocabulary for candidate words. A good heuristic is to only select words that start with the same letter and have a length difference of at most 1.
Step 5: For each candidate, calculate the Edit Distance (Levenshtein distance) which counts how many insertions, deletions, or substitutions are needed to match the words.
Step 6: Sort the candidates from lowest edit distance to highest.
Step 7: Return the top 3 closest matches as suggestions.
