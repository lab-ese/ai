import heapq

def route_finder(start_city, goal_city, graph, heuristics):
    pq = []
    heapq.heappush(pq, (heuristics[start_city], start_city))
    visited = set()
    
    while pq:
        h, current_city = heapq.heappop(pq)
        
        if current_city == goal_city:
            return "Destination Reached"
            
        visited.add(current_city)
        
        for neighbor in graph[current_city]:
            if neighbor not in visited:
                heapq.heappush(pq, (heuristics[neighbor], neighbor))

2. ALGORITHM IN STEPS:
Step 1: Define a graph of cities. Each city has a pre-calculated heuristic value (straight-line distance to the destination).
Step 2: Initialize a Priority Queue (PQ) and push the start city along with its heuristic.
Step 3: Pop the city with the lowest heuristic from the PQ.
Step 4: If the city is the destination, stop and return success.
Step 5: Mark the city as visited.
Step 6: For each connected neighbor of the current city, check if it's unvisited.
Step 7: If unvisited, push it to the PQ along with its predefined heuristic value.
Step 8: Repeat from Step 3.
