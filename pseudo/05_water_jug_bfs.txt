from collections import deque

def water_jug_bfs(cap_x, cap_y, goal):
    queue = deque([(0, 0)])
    visited = set()
    
    while queue:
        x, y = queue.popleft()
        if x == goal or y == goal:
            print("Goal Reached")
            return
            
        if (x, y) not in visited:
            visited.add((x, y))
            queue.append((cap_x, y))        # Fill X
            queue.append((x, cap_y))        # Fill Y
            queue.append((0, y))            # Empty X
            queue.append((x, 0))            # Empty Y
            queue.append(pour(x, y, cap_y)) # Pour X to Y
            queue.append(pour(y, x, cap_x)) # Pour Y to X

2. ALGORITHM IN STEPS:
Step 1: Initialize a queue with the initial state (0, 0).
Step 2: Initialize an empty 'visited' set.
Step 3: Dequeue the front state from the queue.
Step 4: Check if the state satisfies the goal condition. If yes, stop (this path is guaranteed to be the shortest).
Step 5: If the state is not in the visited set, mark it as visited.
Step 6: Generate all valid next states (Fill, Empty, Pour).
Step 7: Enqueue all these new states to the back of the queue.
Step 8: Repeat from Step 3 until queue is empty.

Capacities: X=4, Y=3. Goal=2.

Dequeue (0,3). Add neighbors: (3,0)[Pour Y->X]. Queue=[(4,3), (1,3), (3,0)]
... BFS explores level by level ...
Eventually dequeues (0,2). Enqueues (2,0).
When (2,0) is dequeued, Goal is Reached! BFS finds the optimal shortest sequence.
